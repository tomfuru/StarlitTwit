*概要
自分の好きな機能が入ったTwitterClientを作ろうと思い個人的に作り始めたクライアント．
ひと通りのことはできるはず．
気づいたらメモして気が向いたら直していますがところどころバグが放置されている可能性が・・・

*必要環境
.NET Framework4.0が必要です．
「アプリケーションの初期化に失敗しました」等のエラーが起動して出る場合は.NET Framework4.0がインストールされているか確認してください．

.NET Framework 4.0 Webインストーラー
http://www.microsoft.com/downloads/ja-jp/details.aspx?FamilyID=9cfb2d51-5ff4-4491-b0e5-b386f32c0992

*インストール/アンインストール
必要ありません．設定ファイル等も全てexeのあるフォルダの下に置かれます．不要になったらフォルダごと削除してください．

*始めに
初めて起動した時はメニューの「機能」→「認証」からユーザー認証を行ってください．

*機能
・Twitterのタイムラインの表示・投稿
 - 投稿ではURL短縮も可能(現在bit.ly/j.mpのみ対応)
 - ホーム/リプライ/発言履歴/ダイレクトメッセージ
  * UserStream利用可
 - 自分or他人のリスト
 - 検索
 - 発言は「通常発言(デフォルト背景色:白)」，「自分へのリプライ(デフォルト背景色:赤)」，「他人へのリプライ(デフォルト背景色:青)」，「リツイート(デフォルト背景色:緑)」で表示されます(背景色・フォント設定可能)．
・タブで色々なタイムラインを管理
 - Home,Reply,History,Directは移動不可能だが他のタブはドラッグ&ドロップで移動できます
・リストの作成・編集・削除（メニュー：プロフィール→自分のリスト）
・プロフィールの変更（メニュー：プロフィール→自分のプロフィール）
・ユーザー検索（メニュー：機能→ユーザー検索）
・指定時刻発言取得（メニュー：機能→時刻を指定して発言取得）

・発言の一部選択等がテキスト部分ダブルクリックで可能
 - URL,ハッシュタグ,@ユーザー名等があればそこからURL表示/ハッシュタグのタブ・ユーザーのタブ追加が可能
・ReplyのToolTip表示
 - デフォルトで青色の発言(誰かの発言へのリプライ)の発言部分にマウスを置くとその元発言を取得していれば表示される
  * もし(取得していないものも含む)会話全てを見たいのなら右クリックメニューの「会話を表示」で可
 - 発言をクリックすると表示されません，注意．
・画像サムネイル表示
 - 発言をクリックして，そこに画像へのURLがあれば，ロードしてサムネイル表示する
 - 現在twitpic,photozou,yFrog,img.ly,movapic,plixi,ow.ly,ついっぷるフォト,instagramに対応
 - 複数画像がある場合は一定間隔(デフォルト3秒)ごとに切り替わります
 - もしサムネイルが残ってしまったらそのサムネイルをクリックしてください(要修正・・・)．

*仕様
・ヘッダ/フッタは引用/リプライ/DMにはつきません
・同じ設定ファイルで二重起動はできません

*設定
・設定ファイル(Settings.dat)はただのxmlファイルなのでテキストエディタでいじることもできますが変にいじると読めなくなりますので注意．
・取得設定
 - タイムライン・リプライ・履歴・ダイレクトメッセージのタブの取得数・取得間隔を設定可能
 - 自分のプロフィールの取得間隔を設定可能
・表示設定
 - ツイートの表示フォーマットを設定可能
 - 引用の種類を設定可能
 - アイコンの表示有無，サイズを設定可能
・フォント・色設定
 - 通常発言，自分へのReply，他人へのReply，リツイートそれぞれについて背景色とフォントを設定可能
・サムネイル設定
 - 画像サムネイルの表示有無が設定可能
 - 画像が複数ある時の切り替え間隔を設定可能
 - 各サイトにおけるサムネイルの大きさ/表示有無を設定可能
・UserStream設定
 - 起動時にUserStreamを開始するか設定可能
 - UserStreamでAllReplies(フォロワー→他人,他人→フォロワーのリプライも取得する設定)を有効にするか設定可能
 - イベントがきた時にタスクトレイバルーンに表示するか設定可能
 - UserStreamの開始時ログ表示設定
・フィルタ設定
 後述
・その他設定
 - ヘッダ・フッタを設定可能
 - 確認ダイアログの有無を設定可能
 - Replyのツールチップの深さ設定
 - リプライ・DMのバルーン表示を設定可能
 - ウェブブラウザの設定
  * デフォルトに設定されたウェブブラウザ以外を使用したい時は，使用ウェブブラウザに実行ファイルのパスを設定

* 見つかっているバグ
・画像のサムネイル表示がうまくいかないことがある
・検索で20件までしか表示されない(これはTwitter側のバグらしい)
・その他いっぱい

*要望等
「これうざいなぁ」っていうバグがあればメールやTwitter(下記)で教えて頂ければできるだけ早く(やる気と時間があれば)対処しようかと思います．
「こういうのがあればいいな」という要望は，可能であれば考えてみたいと思います．

*引用等
様々なネット上のソースコードを参考/引用させて頂きました．引用しすぎていたので並べるのが面倒ですが，
ソースコード上には参考/引用した情報があるURLを載せています．
もし問題があればTwitterかメール(下記)で連絡して頂ければ対処致します．
